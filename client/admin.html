Cài đặt chung

Phân cách bằng dấu phẩy

Thông tin kỹ thuật

Hosting: Docker + Tailscale Funnel
Database: PostgreSQL 16 + PostGIS 3.4
Cache: Redis 7 (Alpine)
Reverse Proxy: Nginx (Alpine)
Bản đồ: Leaflet.js + OpenStreetMap + OpenTopoMap
Tọa độ: PostGIS (WGS84 - EPSG:4326)
Frontend: HTML + CSS + JavaScript
Đa ngôn ngữ: i18n Database (2,574 keys)
Xác thực: JWT + MFA (TOTP)
Node.js: v20 (Docker) / v24 (Dev)
